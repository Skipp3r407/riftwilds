# Riftwilds 2D Art Style Guide

Original art only. Do **not** copy Hatchling Eggs, Pokémon, Axie, Neopets, or any other game’s creatures, UI, or palette.

## Illustration rules

- Polished **2D fantasy** game art
- Clean dark outlines, layered cel shading
- Soft upper-left lighting
- Strong readable silhouettes
- Subtle pixel influence only (not full pixel-art UI)
- Transparent backgrounds for items/icons
- No watermarks, no baked-in text on item art
- No photorealism, no modern firearms
- Atmosphere: rift cracks, soft aurora glows, moss/ember/tide accents

## Wallpaper rules

- Full-bleed, edge-to-edge
- Dark navy dominant; cyan + amber glows
- Soft plus-grid or faint rift-line texture OK
- No busy UI chrome baked into wallpaper
- Safe center composition for hero text overlay

## Item rules

- Centered subject, 10–15% padding
- Rarity only via optional subtle aura (not text)
- Affinity color in material / glow, not as a sticker
- Inventory art and icon share silhouette
