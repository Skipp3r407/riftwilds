# UI Raster Assets

Prefer CSS panels. Raster only for:

- Rarity frames (common → celestial)  
- Decorative modal corners  
- Affinity icons (see affinity pipeline)  
- Currency icons (demo credits / soft currency)  
- Achievement icons  
- Battle action icons  
- Bond / XP bar endcaps  

**Do not bake text into PNGs.**

Naming: `ui-{component}-{variant}.png`, `badge-{slug}.png`, `ach-{slug}.png`
