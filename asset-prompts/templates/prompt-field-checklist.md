# Prompt Field Checklist

Every asset prompt file must include:

1. Asset name  
2. Asset category  
3. Purpose  
4. Main subject  
5. Pose  
6. Camera angle  
7. Composition  
8. Art style  
9. Lighting  
10. Color palette  
11. Affinity features  
12. Background requirements  
13. Transparency requirements  
14. Resolution  
15. Negative prompt (or link to shared)  
16. File name  
17. Follow-up editing instructions
