## 1. Full clean egg

| Field | Value |
|-------|-------|
| File | `egg-wild-full.png` |
| Size | 1536 × 1536 |
| Purpose | Master reference / hero render |

**Positive prompt:**

```
Create a completely original fantasy egg for the game Riftwilds, Wild class. The egg is an organic oval with a slightly uneven natural silhouette — not perfectly symmetrical, not a sphere, not a capsule.

Shell surface: pale warm cream with soft ivory highlights and subtle tan undertones. Irregular brown and soft gray speckles scattered naturally across the shell like river-stone markings. Fine matte texture with gentle porous spots. A faint warm inner glow pulses softly beneath the surface — barely visible, suggesting untamed life energy without revealing any creature inside.

Style: polished original 2D fantasy illustration with subtle pixel-art influence, clean dark outlines, layered cel shading, premium mobile-game rendering. Lighting from upper left with soft rim light on the right edge. No environment, no pedestal, no text, no logo.

Center the egg with at least 10% transparent padding on all sides. Transparent background. The design must be completely original and must not resemble any Poké Ball, capture capsule, branded toy egg, or copyrighted game item.
```
