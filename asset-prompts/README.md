# Riftwilds asset prompts

Prompt libraries for original 2D fantasy art. Do not reference other games or copyrighted IP.

## Directories

- `pets/` — species, hatchlings, evolutions, emotes
- `eggs/` — egg types and hatch frames
- `evolutions/` — evolution stage sheets
- `npc/` — mentors, merchants, keepers
- `weapons/` — claw/tail/horn/relic gear
- `armor/` — head/chest/back/paw/wing/barrier
- `potions/` — care and battle bottles
- `abilities/` — VFX key art
- `materials/` — gathering nodes and craft mats
- `furniture/` — homestead décor
- `regions/` — map and biome key art
- `bosses/` — World Rift bosses
- `ui/` — panels, frames, icons
- `effects/` — affinity VFX, hatch FX

Generated raster/SVG placeholders live under `public/assets/`.
