# ZERO_COST_CARD_DESIGN.md

**Authority:** Rules v2.1.0 · `src/game/tcg/rules/zero-cost-design.ts`

## Pool target

**5–10%** of the launch combat pool as carefully balanced 0-cost utilities.  
v2.1 ships **24** collectible 0-costs ≈ **6.9%** of launch combat cards.

## Allowed

- Common / Uncommon
- Tiny companions: atk ≤ 1, hp ≤ 2, **no** Charge/Rush/Swift/…
- Spells: heal ≤ 1, damage ≤ 1, small +Health braces
- Tags: Starter, Utility, Tempo, Support (as appropriate)

## Forbidden

- Legendary / Mythic / Ancient / Founder
- Evolutions
- Board wipes, draw, Energy gain (Rift Spark token is the exception)
- Large bodies, Echo loops, game-ending combos

## Templates

| Card | Role |
|------|------|
| Glowbug | 1/1 nature starter body |
| Pocket Scout / Plaza Scout | 1/1 scout (no Charge) |
| Morning Dew / Festival Cheer | Heal 1 |
| Rift Whisk / Ember Pinch | Deal 1 |

## Deck anti-abuse

`maxZeroCostPerDeck = 4` (hard legality).
