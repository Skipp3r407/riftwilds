# Battle UI

**Component:** `src/components/tcg/rift-battle-board.tsx` · route `/tcg/battle`  
**Rules panel:** `/tcg/rules`

Must show: Keeper HP, Rift Energy, deck/hand counts, defeated/exile, phase, turn clock, Front/Back slot counts, Commander, readiness (Rest/Ready), Rules button.

Field tags mark Front (F) vs Back (B). Practice Board keeps the battle-console command desk UX.

**Mobile / tablet:** dedicated viewport classes — see [MOBILE_UI_GUIDELINES.md](./MOBILE_UI_GUIDELINES.md), [TOUCH_CONTROLS.md](./TOUCH_CONTROLS.md). Phone landscape is preferred; desktop Focus Mode stays flagship.
