# EARLY_GAME_BALANCE.md

## Goals

- Early turns feel interactive without mindless speed.
- Dead opening hands rare.
- Free cards exist but cannot flood decks.

## Levers (v2.1)

| Lever | Setting |
|-------|---------|
| Turn-1 Energy | 2 (unchanged) |
| 0-cost pool | ~7% launch; utility only |
| Deck 0-cost cap | 4 |
| Opening-hand shape | On |
| Mulligan | Keep / Partial / Full |
| AI | Curve-out tempo |

## Non-goals

- Do not raise Energy to “fix” bricks.
- Do not add 0-cost draw / wipes / legendaries.
