# Community Events & Festivals

Seasonal Live World / festival participation can grant **Credits, cosmetics, and occasionally funded prizes**. Never framed as income.

## Paths on Rift Exchange

- `world_events` → `/live-world`
- `community_voting` → `/treasury` (grant vote stubs)
- `bounties` → Exchange anchor (scaffold)
- `weekly_challenges` → `/quests`
- `guild` goals → `/marketplace/guild` + `/guilds`

## Funding

Event prize bands appear only when a treasury / sponsor bucket is **actually funded** and announced. Blank UI means unknown — not zero forever, and not a promise.
