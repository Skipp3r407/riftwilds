# Riftwyrm

> **Status:** CONCEPT — design docs + placeholder only. Not in production species runtime.  
> **Batch:** 2026-07-18 original concept roster

## 1. Creature name
Riftwyrm

## 2. Species title
Gateway Coil Elder

## 3. Personality
Ancient, slow-speaking mentor; coils around Gateways as a living seal.

## 4. Primary affinity
Celestial / Ancient

## 5. Native region (design)
Celestial Rift

## 6. Body structure
Long soft-scaled coil with gateway-ring segments and feathered rift mane (not Western dragon wings).

## 7. Unique silhouette description
Segmented gateway-coil with ring vertebrae — deliberately anti-franchise-dragon (no bat wings, no fire breath meme).

## 8. Main colors
Midnight indigo, gateway cyan, soft gold rings, astral cream mane

## 9. Special features
- Ring vertebrae that echo active Gateways
- Feathered rift mane (soft, not spiked)
- Eyes like distant portals
- Voice like layered choir (lore)

## 10. Signature ability (design)
Seal Coil — stabilize a rift node temporarily

## 11. Gameplay role
Legendary / story elder — not a starter

## 12. Naming notes
Kept Riftwyrm name with anti-dragon silhouette rules
