# Aetherspark

> **Status:** CONCEPT — design docs + placeholder only. Not in production species runtime.

## 1. Creature name
Aetherspark

## 2. Species title
Skyfract Emberling

## 3. Personality
Curious, jittery-bright, loyal once trusted; leaves ozone-sweet static when happy.

## 4. Primary affinity
Storm

## 5. Native region (design)
Stormspire Peaks

## 6. Body structure
Compact mustelid-otter hybrid with sail-fin ears and a filament tail that sheds soft skyfract sparks (not lightning bolts).

## 7. Unique silhouette description
Low runner with two tall sail-ears and a trailing spark-ribbon; reads as storm-weasel, not electric rodent.

## 8. Main colors
Slate violet, pale cyan, warm cream belly, soft gold filament tips

## 9. Special features
- Sail-fin ears that fold flat in rain
- Skyfract filament tail (ribbon, not bolt)
- Cheek vents that puff ozone mist
- Paw pads with faint constellation freckles

## 10. Signature ability (design)
Ribbon Arc — a curved spark-ribbon that marks safe footing

## 11. Gameplay role
Scout / early companion

## 12. Naming notes
Improved name vs generic 'Spark'; avoids Commonspark / Sparkmoth collision by aether + ribbon motif
