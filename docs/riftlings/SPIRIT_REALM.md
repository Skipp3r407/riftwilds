# Spirit Realm

Rescue instance entered when recovering Downed Riftlings. Not a 13th overworld travel region — see `INSTANCE_CONTENT_PACKS` / `SPIRIT_REALM_CONTENT_PACK`.

## Features (stub + content)

- Floating lantern islands, ancient temple hub, light bridges
- Corrupted grove / nightmare echo danger zones
- Puzzle shrine, memorial overlook
- NPCs: Spirit Keeper, Ancient Guardian, Memory Weaver, Soul Blacksmith, Ancestor Council, Dream Walker, Lost Child, Ghost Merchant
- Unique rescue quest kinds (memories, escort, cleanse, nightmare, puzzle, bridge, ancestors, fragments)

## Play

1. Down a Riftling (or wait for critical care)
2. Open Spirit & Recovery → Spirit Realm Rescue Quest
3. Complete quest steps via `/api/pets/[id]/recovery`
4. Companion returns; equipment preserved
