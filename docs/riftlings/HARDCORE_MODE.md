# Hardcore Mode

**Strictly opt-in.** Normal gameplay never permanently kills Riftlings.

## Enable flow

1. Red warning: “This Riftling may be permanently lost.”
2. Acknowledge warning checkbox
3. “I understand…” checkbox
4. Optional typed confirm: `HARDCORE`
5. Server stores `hardcore.enabled` on the spirit record

## Where permadeath applies

Hardcore Expeditions · Hardcore PvE · Ironman · Special Events · Extreme Dungeons

Never: standard Live World, care neglect alone, casual Arena (without Hardcore flag).

## After loss

Creates a Memorial + unlocks Memorial Garden. Equipment ownership remains with the account for memorial display; the Riftling is not marketplace-listable.
