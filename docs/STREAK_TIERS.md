# Streak Tiers

## Daily streak

- Consecutive UTC days with eligible check-in.
- **Miss resets daily streak only** — weekly / monthly / hours / seasonal / event counters persist.

## Tier thresholds

| Tier | Min daily streak |
|------|------------------|
| Bronze | 0 |
| Silver | 3 |
| Gold | 7 |
| Platinum | 14 |
| Diamond | 30 |
| Mythic | 60 |
| Legend | 100 |

## Storm reward weight boosts

| Tier | Boost |
|------|-------|
| Bronze | base |
| Silver | +5% |
| Gold | +10% |
| Platinum | +18% |
| Diamond | +25% |
| Mythic | +30% |
| Legend | +35% |

Boosts improve odds — they never guarantee the best reward.

## Milestones (once each)

7 / 30 / 60 / 90 / 180 / 365 / 500 / 1000 days — titles, badges, cosmetics, Loyalty Tokens, bounded Credits.
