# Companion ↔ card integration

Pipeline: hatch → companion → card (`rotr-comp-*` / `rotr-c-*`) → Codex family → deck builder.

- Companion **level / cosmetics / NFT / token** must not modify ranked combat stats.
- Competitive power uses migrated base card defs only (`powerMode: competitive`).
- Collection mode may show finishes and binder flair.
