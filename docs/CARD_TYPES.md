# CARD_TYPES.md

**Authority:** [RULEBOOK.md](./RULEBOOK.md) · `card-categories.ts`

## Combat-playable

| Category | Notes |
|----------|-------|
| Companion | Units; may receive first-companion discount |
| Evolution | Count toward creature minimum |
| Spell | Slow/fast/reaction speeds |
| Equipment | Needs friendly unit |
| Relic / Terrain / Trap | Support slot bucket |

## Not combat

Food, medicine, materials, care tools, quest props — **Inventory / Companion Care only**.

## Tokens

e.g. Spark Fragment, Rift Spark — special; not constructed collectibles (Rift Spark is match-granted).
