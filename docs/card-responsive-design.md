# Card responsive design

- Progressive disclosure reduces clutter on thumb/hand/field.
- Field overlay is separate from the art plane so live HP can change without re-rasterizing art.
- `loading="lazy"` + `decoding="async"` on master template images.
- Collection list remains capped; binder virtualization is a follow-up for 5k+ catalogs.
- Touch: inspect modal for detail; field tap for play affordances (existing battle board).
