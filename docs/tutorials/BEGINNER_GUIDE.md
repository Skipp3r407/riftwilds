## How to play the path

1. Complete interactives (WASD gate, checklists, click drills).  
2. Pass quizzes (70%+) where present — especially Credits vs SOL.  
3. Mark complete; claim one-time graduate rewards on lesson 30.  
4. Optional: enter Live World and repeat movement/interact for real.

## Graduate rewards

Small one-time badge / title / token credits only — not a farmable faucet.
