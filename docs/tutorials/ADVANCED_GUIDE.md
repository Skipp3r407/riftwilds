# Advanced Path Guide

Filter **Advanced** or **Curricula** in the Academy left rail.

## Mastery tracks (`adv-*`)

| ID | Topic |
| --- | --- |
| `adv-pvp` | PvP fundamentals (no wagering) |
| `adv-ranked` | Ranked ladder + disclosures |
| `adv-guild-wars` | Guild Wars coordination |
| `adv-economy` | Faucets, sinks, treasury — SOL optional |
| `adv-trading` | Marketplace integrity / scam avoidance |
| `adv-crafting` | Optional Ember Forge crafting |
| `adv-farming` | Public farm + homestead |
| `adv-exploration` | Fog, hidden areas, gather |
| `adv-combat` | Turn mastery → Combat Scholar |
| `adv-synergies` | Affinity / passive combos |
| `adv-team` | Team building & coverage |
| `adv-bosses` | Boss telegraphs |
| `adv-restoration` | Multi-step restoration ops |
| `adv-professions` | Optional specializations |
| `adv-dailies` | Daily rotations without FOMO |
| `adv-tournaments` | Brackets & spectating |

## Curricula (`cur-*`)

| ID | Academy |
| --- | --- |
| `cur-riftling` | Riftling sheets & bond |
| `cur-pet-care` | Care loops |
| `cur-combat` | Combat Academy practice stubs |
| `cur-economy` | Credits-first economy |
| `cur-world-map` | Regions / portals / gateways |
| `cur-npc` | Dialogue & services |
| `cur-ui` | Header, sidebar, HUD, Help |
| `cur-controls` | Full keyboard sheet |

## Achievements

- **Tutorial Graduate** — Beginner Path complete  
- **Combat Scholar** — `cur-combat` + `adv-combat`  
- **Economy Aware** — Credits vs SOL complete  
- Others: Care Student, Cartographer, Social Ready, Quiz Whiz, Yard Veteran  

## Practice stubs

Advanced combat lessons deep-link into Arena Training / Duels / Ranked. Web-page stubs mark complete when Live World cannot drive the drill.
