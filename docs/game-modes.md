# Game Modes

Overrides: `MODE_RULE_OVERRIDES` in `battle-rules-config.ts`  
Format catalog: `src/content/tcg/framework/formats.ts`

| Mode | Notes |
|------|-------|
| Standard / Ranked | Full rules, Second Main on |
| Casual / Practice | Auto-skip Second Main; practice soft opening hand |
| Quick Battle | 20 HP, Energy starts 3, 45s timer, auto Second Main |
| Expanded / Legacy | Broader legality pools |
| Draft / Sealed | Planned; relaxed composition |
| Commander | Slightly higher power-rarity cap |
| PvE / Arena | Teaching-friendly bridges |
| Tutorial | No Rift Spark; long timer |

All ranked paths: normalized competitive stats, no pay-to-win.
