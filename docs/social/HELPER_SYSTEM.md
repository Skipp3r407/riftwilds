# Helper System

Opt-in new-player welcoming role.

## Eligibility

- Minimum account age
- Helper tutorial complete
- Good moderation standing
- Opt-in enabled

## Helpers may

Display badge · answer new-player questions · guide to hubs · join beginner activities · share approved tutorial links · earn **capped** Community Tokens / helper achievements

## Helpers must not

Access private account info · request passwords · control other players · handle finances · promise official support · earn unlimited rewards

## APIs

`GET/POST /api/social-presence/helper`

New players can disable helper contact via privacy (`allowHelperContact`).
