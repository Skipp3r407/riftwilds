# Presence XP

Cosmetic / social progression from verified participation. **No combat power.**

## Earning

Meaningful actions (hub visit, NPC talk, emote, pet care, fish, rest interactions, help, performances, daily tasks, …) with recent engagement signals.

## Caps

| Cap | Default |
| --- | --- |
| Per hour | 180 |
| Per day | 900 |

Diminishing returns apply per activity category in a rolling window. Diversity restores value.

## Levels (Wanderer → Riftwilds Luminary)

Data-driven ranks unlock titles, emotes, frames, furniture stubs, guestbook designs — never weapons or marketplace combat advantages.

## Engagement tiers

| Tier | Name | Rewards |
| --- | --- | --- |
| 0 | AFK | None |
| 1 | Resting | Tiny rested progress |
| 2 | Casually Active | Presence XP + small community rewards |
| 3 | Socially Active | Increased XP, recognition eligibility |
| 4 | Community Contributor | Highest capped rate, helper rewards |

## Never

- SOL for idling
- Combat stats / superior Riftlings from Presence XP
- Rewards for motionless open browser
