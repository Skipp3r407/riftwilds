# Home Social Features

Visits, likes, guestbook, ratings, popularity.

## Visitor actions

Tour · like · rate (1–5) · guestbook note · favorite (stub) · photo (stub)

## Owner privacy

Private · friends/guild/invite (future multiplayer) · public hours (stub). Privacy flag `allowHomeVisits`.

## Anti-abuse

Like farming / multi-account reciprocal visits reduce risk score eligibility. Automated visit loops blocked by anti-AFK + risk scoring.

## Featured homes

Daily/weekly selection factors popularity + legitimacy (stub scores via `listPopularHomes()`).
