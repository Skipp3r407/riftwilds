# Player flow — Battle Hub

## Happy path (free)

1. Sidebar **Play → Rift Battle** (or Play hub quick action)
2. Mode cards appear (Practice, Casual, Ranked, AI, Tournament, Rift Stakes)
3. Pick **Practice** → **Open Practice Board** → duel vs Kael
4. Optional: Deck Atelier / Tutorial from the same tab

## Casual / friends

1. Battle Hub → **Casual**
2. Quick match (Arena), Friends (Social), or private invite board

## Ranked season

1. Battle Hub → **Ranked**
2. Open ladder / leaderboards (still free, no wallet)

## Optional SOL

1. Battle Hub → **Rift Stakes** (or `/rift-stakes` redirect)
2. Connect wallet → choose stake → review fee → queue
3. Match room at `/rift-stakes/match`

## Fewer clicks

| Before | After |
|--------|-------|
| Sidebar: Rift Battle **and** Rift Stakes | Sidebar: Rift Battle only |
| Leave Battle to open Stakes | Stay on Battle; switch tab |
| Stakes page links back to Free Practice / Arena | Free modes stay on other tabs |

No functionality removed — destinations are reorganized under the hub and sidebar groups.
