# Battle UI Guidelines

## Do

- Keep the amber / cavern Riftwilds visual language (frame, corners, meters)
- Maximize the center battlefield; treat intel + feed as supporting rails
- Prefer overflow menus for lesser destinations during Focus Mode
- Keep energy costs, hand play affordances, and inventory/deck separation intact
- Use Focus Mode + presets instead of inventing a second art system

## Don’t

- Redesign card art, rarity glow language, or meter iconography
- Remove Match Intel, Event Feed, Practice Board, or Battle Hub mode tabs
- Hide End Turn / Play / Surrender behind obscure chrome
- Let Space end-turn fire while the user is typing in an input
- Stretch board cards to fill ultrawide — grow lane space and card *size presets* instead

## In-battle navigation only

Deck · Codex · Settings · Match History · Exit Match

World / market / guild / collection mega chrome stays collapsed under Focus Mode.

## Contrast

Focus Mode slightly darkens the console wash and raises card/unit contrast so damage, HP, and energy remain readable on a larger stage.
