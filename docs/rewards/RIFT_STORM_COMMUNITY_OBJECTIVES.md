# Rift Storm Community Objectives

Shared goals during a storm with a personal contribution threshold.

## Defaults

- Target community score (e.g. 500)
- Personal threshold (e.g. 10)
- Rewards: bounded Credits + Loyalty Tokens + optional cosmetic

## UI fields

- Personal contribution
- Community total / target
- Next milestone (25/50/75/100%)
- Qualified flag
- Time remaining

Milestones fire audit events; personal community reward is claim-keyed once per storm.
