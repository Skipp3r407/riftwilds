# Player Economy (Rift Exchange)

> Entertainment / rewards framing only. **Never** guaranteed earnings, profit, ROI, or passive income.

## Principles

1. Fun holds even if SOL market value is zero.
2. Core progression uses **Credits** and soft systems — wallet optional.
3. Competitive play stays fair — marketplace / Exchange never sell power.
4. Optional SOL rewards come from **value**: skill, competition, creativity, contribution, cosmetic trade — not match grinding.
5. Ranges shown in UI are **illustrative bands**, not promises.

## Hub

- **Rift Exchange** — `/exchange` — dashboard of modular earning methods
- **Player Marketplace** — `/marketplace` — cosmetics / collectibles trade (one earning path)
- **Reward Center** — `/rewards` — claimable vault framing
- **Treasury** — `/treasury` — community buckets & allocation display
- **Creator Hub** — `/creators` — cosmetic / lore offers

## Currency model (summary)

| Layer | Role |
|-------|------|
| Credits | Primary play currency |
| Gold / soft rewards | Quests, challenges |
| Optional SOL | Prestige pools, verified fee shares, promo drops when funded |
| Launch coin | Entertainment / community — **not** an income product |

See also: `docs/economy/CURRENCY_MODEL.md`, `docs/economy/NO_PAY_TO_WIN_POLICY.md`.
