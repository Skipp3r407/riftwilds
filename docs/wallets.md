# Treasury wallets

| Role | Auto-split | Purpose |
|------|------------|---------|
| PROJECT_TREASURY | No | Single landing for all revenue |
| DEVELOPMENT | 35% | Engineering, audits, build |
| MARKETING | 20% | Campaigns, outreach |
| TOURNAMENT | 15% | Treasury-funded events/prizes (not player escrow) |
| COMMUNITY | 10% | Events/grants — never holder autopay |
| CREATOR | 10% | Creator hub budgets |
| LIQUIDITY | 5% | DEX / MM reserves |
| EMERGENCY | 5% | Incident response |
| OPERATIONS | Manual | Hosting/support (not in default %) |
| COLD_STORAGE | No | Long-term cold |
| BACKUP | No | Recovery staging |
| CUSTOM | Optional | Unlimited admin-added destinations |

Addresses default to `*_COMING_SOON` until configured in `/admin/treasury`.
