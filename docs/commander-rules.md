# Commander Rules

- Exactly **one** Keeper Commander per deck
- Not shuffled into the main deck
- Visible beside Keeper HP
- Defines primary (and optional secondary) element identity
- Active power: typically 2–4 Energy, once per turn (engine flag `powerUsedThisTurn` — execution Phase 2)
- Passives / ultimates wired from hero content in Phase 2

Competitive fairness: Commander cosmetics never alter ranked stats.
