# AAA UI Art Prompts — Riftwilds

High-end illustration prompts for UI chrome. Original dark-fantasy · ancient magical civilization. **No Pokémon / franchise frames.** Generate via existing `npm run assets:generate:ui` pipeline when ready (Phase D).

## Global style lock

```
Dark fantasy UI illustration for Riftwilds TCG. Obsidian stone, aged marble, gold filigree,
arcane cyan glass, warm amber hearth light. Handcrafted materials, subtle particle dust,
cinematic depth. No flat UI, no generic purple neon, no plastic game HUD, no copyrighted
frames or logos. 2.5D presentation plates, readable at HUD scale.
```

---

## 1. Library desk banner (`ui/tcg/library-desk`)

```
Wide cinematic banner of an ancient Riftwilds library desk: open leather Collection Book,
floating ember motes, marble columns with gold filigree, soft cyan riftlight through
stained glass. Top-down slight angle, empty center for title type. 16:5 aspect.
```

Refined Riftwilds brand mark: rift shard + hearth ember, gold + cyan, works on dark HUD

## Delivery notes

- Prefer WebP + transparent PNG where chrome overlays art.
- Keep contrast high enough for HUD readability.
- Foil / goldleaf / crystal finishes stay **cosmetic** in copy and art.
- Register new paths in the asset pipeline before shipping to production.
