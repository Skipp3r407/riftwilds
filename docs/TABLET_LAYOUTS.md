# Tablet Battle Layouts

## Tablet portrait (`tablet-portrait`, ≈768–1024)

- Board-dominant center stage
- Hand dock bottom (larger fan / `tablet-fan`)
- Intel + Feed as collapsible drawers
- Swipe L/R for feed/intel; floating dock + FAB

## Tablet landscape (`tablet-landscape`, ≈1024–1366)

- Centered console (`max-width` clamp)
- Larger hand cards
- Side panels collapsible (narrow rails when collapsed)
- FAB for Intel / Feed shortcuts
- Floating action dock replaces the dense desktop command bar

## Desktop boundary

Above 1366 landscape (and always ≥1920), flagship desktop Focus Mode remains; tablet chrome (dock/FAB) is hidden.
