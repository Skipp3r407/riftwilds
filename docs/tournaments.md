# Rift Arena Tournaments

## Phase order

1. **Free / Credits brackets** — no wallet
2. Sponsored predetermined prizes (legal review; `SPONSORED_PRIZES_ENABLED` off)
3. Optional SOL entry only behind `SOL_TOURNAMENTS_ENABLED` + Arena SOL flags (all default OFF)

## Rules

- Skill brackets only

- Anti-collusion before any SOL pool
