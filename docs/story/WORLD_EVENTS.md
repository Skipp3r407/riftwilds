Mechanical event keys already exist on region packs (ashfall, tidefair, chorus, tempest, etc.). This doc supplies **story meaning** so events feel like history moving, not only loot tables.

## Campaign world events (one-shots)

1. **First Stone Chorus** — Five Gateway Stones sing the same note; Solen opens a temporary Codex wing.
2. **Red Ledger Night** — Serae’s seals fail for one hour; players may copy or return pages (moral fork).
3. **Conductor’s Parade** — Alloy parade offering free repairs that quietly tag Riftlings (Pip warning).
4. **Shepherd’s Quiet** — Void hush muffles Commons music; Mira’s eggs go still until player acts.
5. **Riftstone Cartography** — Riftstone shows a continent that doesn’t exist yet (expansion teaser).

## Living world clock ties

Decade systems (`TEN_YEAR_EXPANSION_PLAN.md`) already have festivals/disasters scaffolds. Narrative rule:

- Disasters should threaten **habitats and bonds**, not punish with pay-to-skip SOL.
- Festival rewards: Credits, care items, cosmetics, Codex scraps — never “investment” framing.

## Gossip & reputation during events

Events seed regional gossip faster (shorter hop). Witnesses of Compact crimes during festivals raise notoriety harder — public eyes.
