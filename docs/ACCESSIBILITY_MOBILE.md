# Accessibility — Mobile Battle

Settings → Battle Layout → **Mobile & accessibility** (also Esc menu).

| Toggle | Effect |
|--------|--------|
| Large cards | Scales hand/field cards; carousel sizes bump |
| One-hand | Biases floating dock toward the thumb edge |
| High contrast | Stronger borders, meter text, card edges |

## Always on (compact)

- 48×48 minimum interactive targets
- Safe-area padding
- Hand carousel instead of unreadably dense fan on phones
- Detail modal for long-press preview (readable type)

## Focus Mode

Immersive desktop Focus Mode is preserved. Compact viewports still set `data-battle-focus` when Focus Mode is on; site header / mobile nav stay suppressed during the active desk.
