# World / environment art style

- Original Riftwilds IP only — no franchise pastiches.
- Painterly digital illustration, atmospheric depth, readable landmarks.
- World **cards**: wide scenic establishing shot, no text/logos/UI.
- **Overview / loading**: cinematic vista of the same region, slightly wider mood, no text.
- Color: match region identity (ember heat, tide moonlight, grove moss, etc.).
- Avoid flat purple gradients, cream-and-terracotta clichés, and generic stock fantasy castles.
