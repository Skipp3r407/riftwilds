# Asset Naming Conventions

**Pattern:** `lowercase-kebab-case`  
**Separator:** `-`  
**Extensions:** `.png`, `.webp`, `.svg`, `.json`

## Creatures

```
creature-{slug}-profile.png
creature-{slug}-card.png
creature-{slug}-icon.png
creature-{slug}-silhouette.png
creature-{slug}-battle-idle.png          # or sheet: creature-{slug}-battle-idle.png
creature-{slug}-battle-attack-basic.png
creature-{slug}-overworld-walk-down.png
creature-{slug}-evolution.png
creature-{slug}-variant-{key}-profile.png
```

## Eggs

```
egg-{class}-full.png
egg-{class}-icon.png
egg-{class}-idle.png
egg-{class}-wobble-left.png
egg-{class}-wobble-right.png
egg-{class}-crack-01.png … crack-03.png
egg-{class}-energy-leak.png
egg-{class}-shell-separate.png
egg-{class}-hatch-burst.png
egg-{class}-open-shell.png
egg-{class}-fragments.png
egg-{class}-silhouette-reveal.png
```

## Other

```
affinity-{name}-icon.png
item-{slug}.png
item-{slug}-icon.png
effect-{slug}.png
region-{slug}-map.png
region-{slug}-battle-bg.png
ui-{component}-{variant}.png
badge-{slug}.png
ach-{slug}.png
```

## Sheets / atlases

```
creature-{slug}-battle-sheet.png
creature-{slug}-battle-atlas.json
creature-{slug}-overworld-sheet.png
creature-{slug}-overworld-atlas.json
egg-{class}-hatch-sheet.png
egg-{class}-hatch-atlas.json
```
