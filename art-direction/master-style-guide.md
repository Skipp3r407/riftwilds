# Riftwilds — Master Art Style Guide

**Project:** Riftwilds · **Creatures:** Riftlings  
**Style:** Polished 2D illustrated pixel-art hybrid for a premium mobile-first browser game.

## Identity

Cute, collectible, expressive, and battle-ready — never childish, never overly aggressive. Fantasy creatures + magical nature + sci-fi portals + elemental energy + classic 2D adventure readability.

## Must combine

- Clean modern 2D digital illustration  
- Detailed pixel-art game sprites (overworld / UI icons)  
- Soft fantasy lighting  
- Bold readable silhouettes  
- Expressive faces  
- Slightly exaggerated proportions  
- Smooth animated VFX  
- Bright elemental energy on dark atmospheric scenes  

## Consistency rules

| Property | Rule |
|----------|------|
| Line thickness | Medium dark outline; slightly thicker on outer silhouette |
| Lighting | Upper-left key light; soft rim on elemental accents |
| Eyes | Large, readable, 2–3 highlight layers; personality first |
| Shading | Layered cel / soft painted bands; avoid muddy gradients |
| Outline | Clean, continuous; break only for glowing energy |
| Texture | Controlled — fur/scales/bark readable at profile size, simplified at overworld |
| Saturation | High but hierarchical; 2–4 dominant colors per subject |
| Sprite scale | Same species family shares comparable on-screen height |
| Animation timing | Idle ~100–160ms; attacks 65–85ms; sleep slower |

## Color direction

Midnight navy · Deep indigo · Glowing cyan · Electric violet · Warm amber · Coral orange · Forest emerald · Crystal blue · Soft cream · Charcoal gray  

Do **not** use every accent in every image. Each region and creature owns a focused palette.

## Originality (non-negotiable)

Do not copy or closely imitate Pokémon, Digimon, Palworld, Neopets, Tamagotchi, Hatchling Eggs, Axie Infinity, or any copyrighted creature, ball, type icon, map, or UI motif.

## Delivery philosophy

- Profile art: illustrated hybrid  
- Battle frames: illustrated with simplified motion-friendly forms  
- Overworld: true pixel-art, nearest-neighbor, strong silhouette  
- UI: prefer CSS panels; raster only for icons, frames, and effects
