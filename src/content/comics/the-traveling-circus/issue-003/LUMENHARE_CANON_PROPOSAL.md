# Lumenhare — Canon Proposal (Issue #3)

**Status:** Proposed for Issue #3; pending Codex species registration.  
**Role:** Support / Illusion / Mobility circus companion of the Lanternveil Traveling Circus.

## Visual design

- Small hare-like Rift companion
- Long luminous ears (soft gold tips)
- Deep indigo fur with gold star markings
- Lantern-shaped tail glow (ember-gold core)
- Large expressive eyes; rapid illusion afterimages when moving
- Original Riftwilds IP — not a franchise mascot

## Abilities (TCG / comic tease)

| Slot | Name | Effect |
|------|------|--------|
| Passive | **Curtain Call** | First time an allied companion would be targeted each round, create an illusion that reduces the effect |
| Active | **Lantern Leap** | Move to another field position; grant one ally Evasion |
| Ultimate | **Grand Illusion** | Temporary illusion copies of allied companions that absorb one attack each |

## Personality / comic voice

- Playful, stage-brave, protective of “backstage” creatures
- Communicates via ear-light pulses, soft thumps, illusion pantomime
- Bonds quickly with Spark through shared resonance language

## Continuity lock

- Markings and indigo/gold palette must not drift across Issue #3 pages
- Scale: roughly Spark-sized (small companion), not a mount
- Associated with Lanternmaster Cael Vesper; not Meridian-owned
